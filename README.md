# URFA — Handmade & Local Brands Marketplace

Mobile marketplace for handmade products, local brands and small sellers.

The app was built around seller storefronts, product publishing, product catalog, buyer communication and order management.

## Technical challenges

### Seller storefronts

The app required seller-specific storefronts where each seller could publish products, manage listings and present their brand.

### Marketplace data structure

The project needed a clear Firestore structure for users, sellers, products, categories, favorites, messages and orders.

### Subscription logic

Seller access and visibility required subscription-style logic, including profile status and paid plan behavior.

### Mobile-first marketplace UX

The main challenge was to keep the marketplace simple enough for small sellers while still supporting catalog, product management and buyer communication.
